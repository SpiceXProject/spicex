About
=====

spicex is a differentiable SPICE circuit solver in Python, leveraging JAX
for automatic differentiation and GPU/TPU acceleration.

The code is open-source and available at:

https://github.com/SpiceXProject/spicex

The documentation is hosted at:

https://spicex.readthedocs.io

The code is developed by `Philip Mocz <https://pmocz.github.io/>`_, launched in 2026.

Community contributions are welcome! Check out `Developing <../pages/developing.html>`_ for guidelines.

Please cite the work if you use spicex in your research; see: `Citing <../pages/citing.html>`_.
